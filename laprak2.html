  1. HTML, CSS, JavaScript

    1. HTML

      2. HTML merupakan singkatan dari Hypertext Markup Language yang dimana merupakan elemen dasar atau kerangka dasar yang digunakan untuk menandai bagian konten untuk ditampilkan di halaman web. HTML digunakan dalam pemrograman web karena HTML itu sendiri seperti bahasa dasar atau utama yang digunakan untuk kita dapat membuat struktur dan konten pada halaman web kita. Yang dimana dapat mengatur elemen-elemen seperti teks, gambar, tautan, formulir, dan tabel. Jadi, HTML ini penting karena menyediakan kerangka kerja di mana semua konten dan elemen lain ditempatkan. Tanpa HTML, tidak ada struktur dasar yang dapat diinterpretasikan oleh browser. Integrasi HTML dalam pemrograman web yaitu HTML sendiri sebagai Struktur Dasar yang dimana HTML digunakan untuk membuat struktur halaman web. Seperti jika kita ingin membuat lampu pintar, maka HTML itu di ibaratkan sebagai tombol awal untuk menyalakan bahkan mematikan lampu.

    2. CSS

      3. CSS merupakan singkatan dari Cascading Style Sheets yang dimana merupakan bahasa yang digunakan untuk mengatur tampilan suatu dokumen yang ditulis dalam markup atau markup language. Terdapat 3 cara untuk pemasangan CSS pada dokumen HTML yaitu : 1. External Style Sheet (file CSS berbeda dari file HTML). 2. Internal Style Sheet (Kode CSS dipasang di dalam tag head HTML). 3. Inline Style Sheet (kode CSS langsung dipasang di tag HTML, tidak direkomendasikan). CSS digunakan dalam pemrograman web karena CSS itu sendiri digunakan untuk mengatur tampilan visual dan tata letak halaman web. Yang Dimana ini mencakup pengaturan warna, font, margin, padding, dan posisi elemen. Jadi, CSS ini memungkinkan pemisahan konten dari desain, yang membuat pengelolaan dan pemeliharaan situs web menjadi lebih mudah. Desain dapat diubah tanpa mengubah struktur HTML. Integrasi CSS dalam pemrograman web yaitu CSS sendiri untuk Desain Visual yang Dimana CSS mengatur tampilan visual dari elemen HTML tanpa mengubah struktur HTML. Seperti jika kita ingin membuat lampu pintar, maka CSS itu akan menambahkan warna dan desain, membuat lampu terlihat cantik dan menarik dengan efek cahaya.

    3. JavaScript

      4. JavaScript sendiri yaitu bahasa pemrograman yang digunakan untuk menambahkan interaktivitas dan fungsi dinamis ke halaman web. Jadi, sama halnya dengan HTML dan CSS, JavaScript juga merupakan salah satu pilar utama pengembangan web. Salah satu library dalam JavaScript yaitu Jquery yang dimana dapat menyederhanakan pemrograman JavaScript. JavaScript digunakan dalam pemrograman web karena javaScript ini bahasa pemrograman yang dapat kita gunakan untuk menambahkan interaktivitas dan fungsi dinamis ke halaman web. Ini mencakup validasi formulir, manipulasi DOM, dan membuat aplikasi web yang kompleks. Yang Dimana javaScript memungkinkan halaman web menjadi interaktif dan responsif, meningkatkan pengalaman pengguna dengan fitur seperti animasi, pemrosesan real-time, dan interaksi pengguna yang kaya. Integrasi JavaScript dalam pemrograman web JavaScript digunakan untuk menambahkan interaktivitas. File JavaScript terpisah dihubungkan dengan tag <|script|> di akhir body HTML, sehingga memastikan bahwa seluruh dokumen HTML telah dimuat sebelum skrip dijalankan. Seperti jika kita ingin membuat lampu pintar, maka JavaScript menambahkan fungsi pintar pada lampu, membuatnya dapat dinyalakan dan dimatikan dengan tombol.
